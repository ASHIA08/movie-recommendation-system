# Movie Recommendation System


A movie recommendation system built with Python and Streamlit. Get personalized and popular movie suggestions through a simple, interactive web interface.


## Features
- Personalized recommendations for any user (from ratings.csv)
- Shows most popular/much-loved movies overall
- "Search for similar movies" by title
- Easy, clickable interface


## How to Run


1. Clone or download this repository
2. Ensure `movies.csv` and `ratings.csv` are in the project folder
3. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
4. Start the app:
    ```
    streamlit run app.py
    ```


## Instructions
- Click the provided buttons to see recommendations or search for similar movies.
- To make the app public, deploy it for free on [Streamlit Cloud](https://streamlit.io/cloud).


## Requirements


- Python 3.x
- `streamlit`
- `pandas`
